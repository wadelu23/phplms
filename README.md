# 圖書館管理系統


* 參考學習資料練習的管理系統，會員資料註冊、書籍狀態、借出歸還、新增圖書、發送訊息等。
* 練習資料庫CRUD，使用者登入資訊SESSION運用，表單內容POST接收，網址參數傳遞。


學生
---
* 學生可註冊會員
* 查詢書本
* 觀看借書紀錄
* 接收訊息

管理員
---
* 新增書籍
* 書籍借出與歸還
* 查看借閱狀況
* 學生註冊核准
* 發送訊息
